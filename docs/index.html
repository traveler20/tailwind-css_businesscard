できること

  • devices
    コーディングスキル

    HTML/CSS/JavaScriptを用いたコーディングはもちろん、gulpやViteを用いてEJSとDart Sassを用いたコーディングも可能です。CSS設計はFLOCSSを用いることが多いです。

  • palette
    デザインスキル

    FigmaやAdobe XDを用いたWEBサイトの設計・デザインが可能です。
    PhotoshopやIllustratorを用いたバナー制作なども担当しています。

  • dns
    サイトの構築・運用

    静的ウェブサイトのWordPress組み込みやWordPressのテーマ化、MovableType.net、Wraptasへの組み込みができます。
    Looker Studioを用いたGA4やGoogle Search Console等のレポートの出力も担当しています。
